Disaster Response Assistant

The Disaster Response Assistant is an application that allows users to simulate and respond to different disasters, both real and fictional.

Locations

The DRA Application allows users to input any location in the US. We have a large backend database containing the the population and population density of over 25,000 locations. However, if a user wishes to put in another location, the location will be searched for via python module geopy and if found, will be used in the simulation.


Disasters

Currently, the application supports several disasters. This disasters include Hurricanes, Earthquakes, Wildfires, Pandemics, and Tornados. Science fiction disasters include Asteroids, Alien Invasion, and the Sun stopping fusion.


Actions

During simulations, the user must choose from the allowed actions in order to advance. The actions allowed include Waiting, Declaring a State of Emergency, Applying for Federal Aid, Applying for State Aid, Ordering an Evacuation, and Ordering to Shelter in Place. Actions corresponding to science fiction disasters include Recruit Bruce Wilis, Declare Anarchy, Declaring Fealty.
