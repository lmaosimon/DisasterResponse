# Contributing

This repository is maintained as a public portfolio project. Small, targeted improvements are preferred over broad rewrites.

## Workflow

1. Open an issue or describe the problem you want to solve.
2. Create a focused branch for the change.
3. Add or update tests when behavior changes.
4. Run `python3 -m unittest discover -s Test -t . -v` before opening a pull request.

## Pull Request Expectations

- explain the scenario being improved
- keep changes scoped to one concern
- note any follow-up work that remains out of scope
